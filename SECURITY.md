## Security Policy

If you believe you have found a security vulnerability in this repository, please **do not** open a public issue.

- Prefer: **GitHub Security Advisories** (private report)
- Alternatively: open an issue with **no sensitive details** and ask for a private channel

This is an open-source research/tooling repository. **No guarantees** are made regarding response time, fixes, or releases. The software is provided **as-is** without warranty; use at your own risk.
